# 图标系统使用指南

## 概述

本文档介绍如何在项目中使用 SVG 图标系统，以替代 Emoji 图标。这个系统确保在所有操作系统（Mac、Windows、Linux）上显示一致的图标。

## 文件结构

```
icons/
├── icons.svg              # SVG 图标集合
├── icons.css              # 图标样式文件
├── SvgIcon.vue            # SVG 图标组件
├── IconLibrary.vue        # 图标库（导出所有图标）
└── ICON_USAGE_GUIDE.md    # 本文档
```

## 快速开始

### 1. 导入文件

在您的 Vue 项目中导入必要的文件：

```vue
<!-- 在 main.ts 或 App.vue 中 -->
<template>
  <div>
    <!-- 在 HTML 中引入 SVG 图标集 -->
  </div>
</template>

<script setup>
import SvgIcon from '@/components/SvgIcon.vue'
</script>

<style>
@import '@/styles/icons.css'
</style>
```

### 2. 在 HTML 中引入 SVG 集合

在 `public/index.html` 的 `<body>` 标签中添加：

```html
<body>
  <div id="app"></div>
  <!-- 引入 SVG 图标集 -->
  <svg style="display: none;">
    <use href="/icons.svg"></use>
  </svg>
  <script src="/src/main.ts" type="module"></script>
</body>
```

或者在 `App.vue` 中：

```vue
<template>
  <div id="app">
    <!-- 您的应用内容 -->
  </div>
  <!-- SVG 图标集 -->
  <svg style="display: none;">
    <use href="/icons.svg"></use>
  </svg>
</template>
```

### 3. 使用图标

#### 基本用法

```vue
<template>
  <SvgIcon icon="icon-search" />
</template>

<script setup>

```

#### 自定义大小

```vue

  <SvgIcon icon="icon-search" size="md" />
```

### 在表格中使用

```vue
<template>
  <table>
    <thead>
      <tr>
        <th>文件名</th>
        <th>类型</th>
        <th>状态</th>
      </tr>
    </thead>
    <tbody>
      <tr v-for="file in files" :key="file.id">
        <td>
          <SvgIcon :icon="getFileIcon(file.type)" size="md" />
          {{ file.name }}
        </td>
        <td>{{ file.type }}</td>
        <td>
          <SvgIcon
            :icon="getStatusIcon(file.status)"
            :class="`icon-${file.status}`"
            size="md"
          />
          {{ file.status }}
        </td>
      </tr>
    </tbody>
  </table>
</template>

<script setup>
const getFileIcon = (type) => {
  const icons = {
    pdf: 'icon-pdf',
    xlsx: 'icon-excel',
    docx: 'icon-word',
    txt: 'icon-file'
  }
  return icons[type.toLowerCase()] || 'icon-file'
}

const getStatusIcon = (status) => {
  const icons = {
    success: 'icon-success',
    error: 'icon-error',
    warning: 'icon-warning',
    info: 'icon-info'
  }
  return icons[status.toLowerCase()] || 'icon-info'
}
</script>
```

## 自定义图标

如果需要添加新的图标，请按以下步骤操作：

### 1. 在 `icons.svg` 中添加新的 symbol

```xml
<symbol id="icon-custom" viewBox="0 0 24 24">
  <!-- 您的 SVG 路径 -->
  <path d="..." fill="none" stroke="currentColor" stroke-width="2"/>
</symbol>
```

### 2. 在 Vue 中使用

```vue
<template>
  <SvgIcon icon="icon-custom" />
</template>
```

## 性能优化

### 1. 使用 CDN 加载 SVG

```html
<svg style="display: none;">
  <use href="https://cdn.example.com/icons.svg"></use>
</svg>
```

### 2. 图标精灵表优化

所有图标已合并到单个 SVG 文件中，减少网络请求。

### 3. CSS 类名缓存

使用 CSS 类名而不是内联样式，提高缓存效率。

## 常见问题

### Q: 图标在某些系统上显示不正确？

A: 确保已正确导入 `icons.svg` 文件，并且 SVG 文件路径正确。

### Q: 如何改变图标颜色？

A: 使用 `color` 属性或 CSS 类名：
```vue
<SvgIcon icon="icon-search" color="#ff0000" />
<SvgIcon icon="icon-search" class="icon-error" />
```

### Q: 如何添加自定义动画？

A: 在 `icons.css` 中添加新的 keyframe 动画，然后在组件中使用：
```vue
<SvgIcon icon="icon-search" class="my-custom-animation" />
```

### Q: 图标太小或太大？

A: 使用 `size` 属性调整：
```vue
<SvgIcon icon="icon-search" size="lg" />
```
