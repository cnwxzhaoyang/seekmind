# 阶段 7 可执行任务列表：OCR 和更多格式支持

本文档从 [《阶段 7 缺口清单》](./43-阶段7缺口清单(2026-06-08).md) 进一步拆解为可以直接排期执行的任务列表。

## 任务列表

### 1. 补齐 XLSX 文本提取

目标：
- 支持 `.xlsx` 文件进入索引与搜索链路。

任务：
- 选择 XLSX 解析方案。
- 实现基础文本提取，不追求单元格样式和复杂表格结构。
- 将 XLSX 加入支持扩展名与解析分支。
- 补充失败提示和状态说明。

验收：
- 用户导入含 `.xlsx` 的目录后，文件能够被成功索引并参与搜索。

### 2. 补齐 EPUB 文本提取

目标：
- 支持 `.epub` 文件进入索引与搜索链路。

任务：
- 实现 EPUB 解包与章节文本抽取。
- 保留章节顺序，转换为可搜索纯文本。
- 将 EPUB 加入支持扩展名与解析分支。
- 补充失败提示和状态说明。

验收：
- 用户导入含 `.epub` 的目录后，章节内容能够被索引和搜索。

### 3. 补齐 RTF 文本提取

目标：
- 支持 `.rtf` 文件进入索引与搜索链路。

任务：
- 选择 RTF 到文本的转换方式。
- 将 RTF 加入支持扩展名与解析分支。
- 补充失败提示和状态说明。

验收：
- 用户导入含 `.rtf` 的目录后，正文能够被索引和搜索。

### 4. 接入图片 OCR 主链路

目标：
- 图片文件能够在索引流程中自动识别文字。

任务：
- 选定支持的图片格式范围，至少覆盖 PNG / JPG / JPEG。
- 在索引阶段接入图片 OCR。
- 将识别结果写回 blocks / chunks / search 可用的数据结构。
- 对 OCR 失败、语言不可用、图片过大等情况做明确提示。

验收：
- 导入图片文件后，图片中文字能够被搜索到。

### 5. 补齐扫描版 PDF 检测闭环

目标：
- 自动识别扫描版 PDF，并进入 OCR 处理链路。

任务：
- 定义扫描版 PDF 的判定规则。
- 实现检测后自动入队 OCR。
- 将 OCR 结果回填到文档块或 OCR 任务表。
- 补齐重跑、失败、恢复等边界处理。

验收：
- 用户对扫描版 PDF 执行 OCR 后，能够通过文字搜索到对应页面。

### 6. 补齐格式支持状态说明

目标：
- 让用户明确知道当前支持哪些格式、哪些格式会降级、哪些格式不可用。

任务：
- 在状态页或设置页增加统一的格式支持说明。
- 列出各格式的支持情况、降级路径和限制。
- 保持中英文国际化。

验收：
- 用户可以直接看到当前格式支持边界。

## 推荐执行顺序

1. XLSX
2. EPUB
3. RTF
4. 图片 OCR 主链路
5. 扫描版 PDF 检测闭环
6. 格式支持状态说明

## 交付原则

- 先保证“可搜索”，再考虑“结构化预览”。
- 先覆盖常见文件格式，再补边缘格式。
- OCR 相关功能优先保证可用性和失败提示，不追求版面还原。
